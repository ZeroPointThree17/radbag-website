Information Collection and Use

Why we ask for Location permission

  • In order to connect any hardware wallet via a Bluetooth connection, we need to enable Bluetooth and Location permission on Android
  • We are using Location permission on Android to send Bluetooth Low Energy messages between the app and the hardware wallet. We will NOT use or share your location at any time.
  • Location permission on Android is requested for any hardware wallet regardless of whether it allows bluetooth connections or not. This is because the scanning mechanism cannot determine ahead of time if the hardware wallet does not support bluetooth.
  • If you do not use a hardware wallet, Location permission is never requested.
  • The iOS version of the app never requests Location permissions.
